# ProjectEuler

## What is Project Euler?
Project Euler is a series of challenging mathematical/computer programming problems that will require more than just mathematical insights to solve. Although mathematics help arrive at elegant and efficient methods, the use of a computer and programming skills is required to solve most problems.
